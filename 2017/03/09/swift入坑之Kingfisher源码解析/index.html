分析一下这段代码

swift的预编译语法和OC一样，这段代码把不同类型中相同的控件通过关键字typealias重命名，这种做法比较常见，跨平台的方案中经常用这种方式来兼容。

关键字 typealias，给系统的类型取别名，相当于OC中的 typedef

有两个宏定义可以关注一下
os(macOS) 和 os(watchOS)
